# محدودیت‌های V1

مواردی که عمداً کامل نشده‌اند یا نیاز به بهبود دارند.

## محصول

| محدودیت | توضیح |
|---------|--------|
| export داده | خروجی Excel/CSV self-service کامل نیست |
| اعلان real-time | بدون WebSocket/push |
| چند workspace همزمان | یک org در session |
| فرم/جدول یکدست | بخشی از UI هنوز یکپارچه‌سازی کامل ندارد |

## پرداخت و کانال‌ها (بازار ایران)

| محدودیت | توضیح |
|---------|--------|
| درگاه پرداخت آنلاین | متصل نیست؛ ثبت دستی پرداخت |
| پیامک/واتساپ خودکار | اتوماسیون محدود |

## دستیار هوشمند

| محدودیت | توضیح |
|---------|--------|
| تاریخچه گفتگو | پایدار نیست |
| قالب پاسخ | plain text |
| وابستگی سرویس | حالت degraded در قطعی |

## فنی

| محدودیت | توضیح |
|---------|--------|
| آپلود روی Vercel | ذخیره‌سازی موقت serverless |
| backup ابری خودکار | زمان‌بندی خارج از repo |
| ممیزی WCAG رسمی | انجام نشده |
