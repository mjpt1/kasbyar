# توسعه محلی

## پیش‌نیاز

- Node.js 20+
- npm
- PostgreSQL 16+ (یا از Docker)
- (اختیاری) Python 3.12 برای سرویس AI

## راه‌اندازی

```bash
cp .env.example .env
npm run setup
npm run dev
```

`setup` معمولاً نصب وابستگی‌ها، generate Prisma و آماده‌سازی اولیه را پوشش می‌دهد.

## دیتابیس

| دستور | کار |
|-------|-----|
| `npm run db:generate` | Prisma Client |
| `npm run db:migrate` | migration توسعه |
| `npm run db:push` | همگام schema (محیط محلی/staging) |
| `npm run db:seed` | داده نمونه |
| `npm run db:reseed` | seed مجدد |

## کیفیت

```bash
npm run lint
npm run typecheck
npm test
npm run verify   # ترکیب رایج قبل از PR
npm run ci       # نزدیک به pipeline
```

## عیب‌یابی سریع

| مشکل | بررسی |
|------|--------|
| خطای DATABASE_URL | `.env` و وضعیت Postgres |
| خطای SESSION_SECRET | حداقل طول در env |
| حلقه redirect ورود | کوکی نامعتبر؛ middleware نباید فقط با وجود کوکی به داشبورد بفرستد |
| AI کار نمی‌کند | `AI_SERVICE_URL` / توکن / اجرای `dev:ai` |
