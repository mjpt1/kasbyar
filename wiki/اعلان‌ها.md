# اعلان‌ها

اعلان‌های داخل اپ در **زنگولهٔ هدر** ذخیره می‌شوند و از اتوماسیون (`NOTIFY_USER`) و بریفینگ روزانه هم پر می‌شوند.

## داخل اپ

| مورد | مسیر / محل |
|------|------------|
| زنگوله | هدر اپ |
| API لیست / خوانده‌شده | `/api/notifications` |
| مدل‌ها | `InAppNotification`, `PushSubscription` |

## Web Push (مرورگر)

برای اعلان مرورگر:

```bash
npx web-push generate-vapid-keys
```

در `.env`:

```
VAPID_PUBLIC_KEY=...
VAPID_PRIVATE_KEY=...
NEXT_PUBLIC_VAPID_PUBLIC_KEY=...   # اختیاری؛ برای کلاینت
VAPID_SUBJECT=mailto:ops@kesbyar.ir
```

| مورد | توضیح |
|------|--------|
| اشتراک | `/api/push/subscribe` |
| Service Worker | `apps/web/public/sw.js` (رویداد `push` و `notificationclick`) |
| بدون VAPID | اعلان داخل اپ همچنان کار می‌کند |
