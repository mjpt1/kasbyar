# شروع سریع

## نسخه آنلاین (بدون نصب)

1. به https://kasbyar.vercel.app بروید  
2. ثبت‌نام کنید، یا با حساب دمو وارد شوید  
3. فضای کاری (سازمان) را انتخاب کنید  

| ایمیل | رمز | نقش |
|-------|-----|-----|
| `demo@kesbyar.ir` | `demo1234` | مالک / دمو |
| `manager@kesbyar.ir` | `demo1234` | مدیر |
| `staff@kesbyar.ir` | `demo1234` | کارمند |
| `super@kesbyar.ir` | `demo1234` | سوپرادمین |

## اجرای محلی

پیش‌نیاز: Node.js 20+، npm، و PostgreSQL (یا Docker).

```bash
git clone https://github.com/mjpt1/kasbyar.git
cd kasbyar
cp .env.example .env
npm run setup
npm run dev
```

| سرویس | آدرس |
|--------|------|
| وب و لندینگ | http://localhost:3000 |
| ورود | http://localhost:3000/login |
| دستیار (اختیاری) | `npm run dev:ai` → پورت `8000` |

### Docker برای دیتابیس و AI

```bash
npm run docker:up
```

جزئیات بیشتر: [توسعه محلی](./توسعه-محلی.md) · [متغیرهای محیط](./متغیرهای-محیط.md)

## دستورات پرکاربرد

| دستور | کار |
|-------|-----|
| `npm run dev` | اجرای وب |
| `npm run build` | build تولید |
| `npm run verify` | lint + typecheck + test |
| `npm run db:migrate` | migration توسعه |
| `npm run db:seed` | داده نمونه |
| `npm run db:reseed` | پاک‌سازی و seed مجدد (محیط غیر production) |
