# دستیار هوشمند

سرویس دستیار عملیاتی جدا از اپ وب اجرا می‌شود (`apps/ai-service` — FastAPI).

## رفتار محصول

| حالت | نتیجه |
|------|--------|
| سرویس در دسترس | بینش و کمک عملیاتی در UI |
| سرویس قطع | هسته CRM/فاکتور کار می‌کند؛ دستیار degraded |

## احراز هویت داخلی

ارتباط وب → AI با توکن داخلی (`AI_SERVICE_TOKEN`) انجام می‌شود؛ نه با session کاربر مرورگر.

## محدودیت‌های V1

- بدون تاریخچه پایدار گفتگو (با refresh پاک می‌شود)  
- پاسخ plain text (بدون markdown غنی)  
- وابسته به استقرار جداگانه سرویس  

جزئیات: [محدودیت‌های V1](محدودیت‌های-V1) · مخزن `docs/AI_INTEGRATION.md`

## اجرای محلی

```bash
npm run dev:ai
```
