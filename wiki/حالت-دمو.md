# حالت دمو (Demo Mode)

حالت نمایش برای فروش، آموزش و پایلوت — با داده نمونه و سناریوهای آماده.

## فعال‌سازی

در `.env`:

```bash
DEMO_MODE=true
NEXT_PUBLIC_DEMO_MODE=true
```

سپس:

```bash
npm run db:reseed
```

مرکز نمایش: `/demo`

## سناریوها

سناریوهای دمو بر اساس بسته صنفی (عمومی، کلینیک، خرده‌فروشی، سفر) مسیر walkthrough و داده نمونه متفاوت دارند. پس از ورود می‌توانید از صفحه انتخاب فضا / دمو سناریو را عوض کنید.

## نکات امنیتی

- در production فقط وقتی آگاهانه برای محیط فروش روشن شود  
- `ALLOW_SEED` در production باید `false` باشد مگر staging کنترل‌شده  
- دمو نباید به داده واقعی مشتری مخلوط شود  

جزئیات در مخزن: `docs/DEMO.md` · `docs/SEED.md`
