# Operator / Admin Quick-Start — Pilot Customer

**For:** OWNER or ADMIN at pilot customer  
**Language:** Persian UI

---

## اولین ورود

1. به آدرس workspace بروید (از تیم کسب‌یار دریافت کرده‌اید)
2. با ایمیل و رمز عبور وارد شوید
3. workspace سازمان خود را انتخاب کنید

---

## کارهای روز اول

| کار | مسیر |
|-----|------|
| دیدن خلاصه کسب‌وکار | `/dashboard` |
| افزودن مشتری | `/customers` → جدید |
| ثبت لید | `/leads` |
| صدور فاکتور | `/invoices` → جدید |
| ثبت دریافت | `/payments` |
| پرسش از دستیار* | `/conversation` |
| دعوت همکار | `/settings` (نقش‌ها) |
| راهنما | `/help` |

\* نیاز به طرح مناسب — اگر غیرفعال است پیام ارتقا می‌بینید.

---

## نقش‌ها

| نقش | دسترسی |
|-----|--------|
| OWNER | همه چیز + billing |
| ADMIN | مدیریت تیم + تنظیمات |
| STAFF | CRM و فاکتور روزانه |

---

## دستیار هوشمند

- در صورت قطعی سرویس، **حالت آفلاین** نمایش داده می‌شود — بقیه اپ کار می‌کند
- تاریخچه مکالمه با refresh پاک می‌شود (محدودیت فعلی)
