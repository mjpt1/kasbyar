# Business Memory

## Purpose
Business Memory stores and retrieves the organization’s knowledge.

## Sources
- Documents
- Contracts
- Invoices
- Notes
- Meetings
- Messages
- Files
- Operational events

## Capabilities
- Semantic search
- Structured retrieval
- Contextual answering
- Timeline reconstruction
- Entity linking

## Principles
- Store provenance
- Preserve source references
- Scope by tenant
- Support future RAG workflows
