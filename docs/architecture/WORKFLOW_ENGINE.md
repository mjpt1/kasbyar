# Workflow Engine

## Purpose
Enable no-code and low-code business automation.

## Building Blocks
- Trigger
- Condition
- Action
- Delay
- Branch
- Approval
- AI decision step

## Requirements
- Visual representation
- Execution history
- Retry support
- Auditability
- Permission-aware steps

## Examples
- New customer -> create invoice -> send notification
- Payment received -> update project -> assign task
- Risk detected -> notify manager -> create follow-up
