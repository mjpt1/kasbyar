# حافظه شرکت (`/memory`)

دانش سازمانی قابل جستجو: اسناد، یادداشت‌ها، رویدادها و خط زمانی.

## قابلیت‌ها

- **ورود (ingest)**: تکه‌تکه کردن متن، embedding، ذخیره
- **جستجو**: بازیابی معنایی روی حافظه سازمان
- **خط زمانی**: مرور رویدادها و قطعات مرتبط

## نحوه استفاده

1. به `/memory` بروید
2. محتوا وارد کنید یا جستجو بزنید
3. نتایج را در تصمیم‌گیری یا دستیار استفاده کنید

## API مرتبط

- `POST /api/memory/ingest`
- `POST /api/memory/search`
- `GET /api/memory/timeline`
