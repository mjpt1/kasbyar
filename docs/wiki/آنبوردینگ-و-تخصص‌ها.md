# آنبوردینگ و تخصص‌ها

اگر `industrySpecialty` خالی باشد، مالک/مدیر به `/onboarding` می‌رود.

1. نام کسب‌وکار  
2. حوزه (`industryPack`)  
3. تخصص (`industrySpecialty`) فیلترشده بر اساس حوزه  
4. هدایت به `/v/{specialty}` یا داشبورد

## عمومی (`GENERAL`)

| تخصص | مسیر |
|------|------|
| کسب‌وکار عمومی | `/v/general-business` |
| فریلنسر | `/v/freelancer` |
| شرکت برنامه‌نویسی | `/v/software-house` |
| مشاوره | `/v/consulting-firm` |
| خدمات آنلاین | `/v/online-services` |
| تولید محتوا | `/v/content-creator` |

API: `/api/onboarding`
