# اعلان‌ها

اعلان‌های داخل اپ در زنگولهٔ هدر ذخیره می‌شوند و از اتوماسیون و بریفینگ روزانه هم تغذیه می‌شوند.

## داخل اپ

- زنگوله در هدر
- API: `/api/notifications`
- مدل‌ها: `InAppNotification`, `PushSubscription`

## Web Push

```bash
npx web-push generate-vapid-keys
```

در `.env`:

```
VAPID_PUBLIC_KEY=...
VAPID_PRIVATE_KEY=...
NEXT_PUBLIC_VAPID_PUBLIC_KEY=...
VAPID_SUBJECT=mailto:ops@kesbyar.ir
```

بدون این کلیدها، اعلان داخل اپ همچنان کار می‌کند. اشتراک: `/api/push/subscribe` — Service Worker: `apps/web/public/sw.js`.
