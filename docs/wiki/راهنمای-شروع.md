# راهنمای شروع

## پیش‌نیازها

- Node.js 20+
- Docker Desktop (برای Postgres و سرویس AI)
- کلید API سازگار با OpenAI (اختیاری برای LLM واقعی)

## راه‌اندازی محلی

```bash
git clone https://github.com/mjpt1/kasbyar.git
cd kasbyar
cp .env.example .env
npm run docker:up
npm run setup
npm run dev
```

| سرویس | آدرس |
|--------|------|
| وب | http://localhost:3000 |
| ورود | http://localhost:3000/login |
| AI (اختیاری) | `npm run dev:ai` → پورت 8000 |

## حساب‌های دمو

| ایمیل | رمز | نقش |
|-------|-----|-----|
| `demo@kesbyar.ir` | `demo1234` | مالک / دمو |
| `manager@kesbyar.ir` | `demo1234` | مدیر |
| `staff@kesbyar.ir` | `demo1234` | کارمند |

در .env برای حالت نمایش:

```env
DEMO_MODE=true
NEXT_PUBLIC_DEMO_MODE=true
```

سپس `npm run db:reseed` و باز کردن `/demo`.

## متغیرهای مهم

| متغیر | نقش |
|--------|-----|
| `DATABASE_URL` | اتصال Postgres |
| `AI_SERVICE_URL` | آدرس FastAPI (مثلاً `http://localhost:8000`) |
| `AI_SERVICE_TOKEN` | توکن داخلی بین وب و AI |
| `LLM_API_KEY` | کلید LLM (OpenAI-compatible) |
| `LLM_API_URL` | پایه API (پیش‌فرض OpenAI) |
| `LLM_MODEL` | مدل (مثلاً `gpt-4o-mini`) |

جزئیات بیشتر: [پیکربندی LLM](پیکربندی-LLM) و فایل `.env.example`.

## اولین قدم‌ها در محصول

1. وارد شوید و فضای کاری دمو را انتخاب کنید.
2. از منوی **هوشمند** به [اتاق فرمان](اتاق-فرمان) بروید.
3. راهنمای داخل اپ: `/help`.
