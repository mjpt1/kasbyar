# پلتفرم و افزونه‌ها (`/platform`)

لایه توسعه‌دهنده: Agent SDK، رجیستری ایجنت‌ها و قابلیت گسترش.

## مفاهیم

- **Agent SDK** (`packages/agent-sdk`): قرارداد ابزار و ایجنت
- **ارکستراتور**: هماهنگی ایجنت‌های دپارتمانی
- **افزونه‌ها**: اتصال قابلیت‌های جدید بدون تغییر هسته

## نحوه استفاده

1. `/platform` را برای وضعیت پلتفرم باز کنید
2. مستندات معماری: `docs/architecture/AGENT_PLATFORM.md`
