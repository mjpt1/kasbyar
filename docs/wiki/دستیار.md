# دستیار (`/conversation`)

چت عملیاتی فارسی برای پرسش از داده کسب‌وکار و پیشنهاد اقدام.

## قابلیت‌ها

- پاسخ بر اساس زمینه عملیاتی (مشتری، فاکتور، وظیفه، …)
- پیشنهاد اقدام با **تأیید انسان** قبل از اجرا
- اتصال به حافظه شرکت و ابزارهای هوش

## نحوه استفاده

1. `/conversation` را باز کنید
2. سؤال بپرسید (مثلاً «فاکتورهای معوق این هفته؟»)
3. اگر اقدام پیشنهادی آمد، قبل از اجرا **تأیید** کنید

## API مرتبط

- `POST /api/conversation`
- `POST /api/conversation/confirm`
