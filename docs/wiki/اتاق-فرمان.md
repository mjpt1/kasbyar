# اتاق فرمان (`/command`)

مرکز فرماندهی روزانه کسب‌وکار: بریفینگ، اولویت‌ها و نمره سلامت.

## چه می‌بینید؟

- **بریفینگ روزانه**: خلاصه وضعیت عملیات، درآمد، وظایف و هشدارها
- **اولویت‌های پیشنهادی**: کارهای مهم روز برای مدیر
- **نمره سلامت**: شاخص ترکیبی سلامت کسب‌وکار (health scores)

## نحوه استفاده

1. از سایدبار بخش **هوشمند** → **اتاق فرمان**
2. بریفینگ را مرور کنید؛ در صورت نیاز صفحه را تازه‌سازی کنید
3. برای جزئیات بیشتر به دستیار، پیش‌بینی یا وظایف بروید

## API مرتبط

- `GET/POST /api/briefing/daily`
- `GET /api/health/scores`
